# ¿Qué es el Risco? 🎲
El Risco es un juego de dados en el que tiras tres dados y puedes decidir a que categoría jugar (por ejemplo, escalera mayor, escalera impar, trío, etc.) Hay 13 rondas (una por cada categoría) y gana el jugador que mas puntos consigue.

# Ejecución 🖥️
Para ejecutar el juego simplemente debes bajar este repositorio, compilar TestPartida y ejecutarlo mínimo con Java 8.

```bash
git clone https://github.com/mariamm99/ProyectoDados && cd ProyectoDados/juegoRisco

javac risco/TestPartida.java

java risco/TestPartida
```
